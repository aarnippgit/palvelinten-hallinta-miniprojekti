Aloitimme ensin testaamalla TeamSpeak serverin asennuksen käsin Debian13 koneelle ja siihen yhdistämisen. Ensimmäiseksi kopioimme TeamSpeak server download sivulta latausosoitteen. Terminaalissa loimme uuden käyttäjän nimeltä teamspeak komennolla sudo adduser teamspeak. Vaihdoimme teamspeak käyttäjälle komenolla su teamspeak ja sirryimme sen kotihakemistoon komennolla cd /home/teamspeak. Jotta lataustiedoston sai purettua, piti asentaa lbzip2 komenolla sudo apt -y install lbzip2. Tämän jälkeen liitimme kopioidun lataustiedoston wget komennon perään ja asensimme teamspeak lataustiedoston. Asennuksen jälkeen tiedosto piti purkaa komennolla tar -xf teamspeak3-server_linux_amd64-3.13.7.tar.bz2. Sitten kopioimme sen teamspeak käyttäjän kotihakemistoon. Tämän jälkeen poistimme alkuperäisen ladatun tiedoston ja siinä luodun kansion. Viimeinen vaihe teamspeak käyttäjällä oli luoda tiedosto .ts3server_license_accepted, joka hyväksyy teamspeak lisenssin. Sitten komennolla exit vaihdoimme takaisin pääkäyttäjälle. Pääkäyttäjällä loimme tiedoston teamspeak servicelle. sudo nano /lib/systemd/system/ts3server.service. Tiedoston luonnin jälkeen startattiin ja otettiin teamspeak käyttöön. sudo systemctl start ts3server. sudo systemctl enable ts3server. TeamSpeak luo automaattisesti admin käyttäjän ja admin tokenin, jonka avulla saat itsellesi admin oikeudet serveriin. Admin käyttäjän salasana ja token piti ottaa talteen komennoilla sudo journalctl -xe -u ts3server | grep password. sudo journalctl -xe -u ts3server | grep token. Viimeiseksi komennolla sudo ufw allow piti avata seuraavat portit 9987/udp, 30033/tcp, 10011/tcp, 41144/tcp.
